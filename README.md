# Оптимизация веб-ресурсов

Курсовой проект по дисциплине "Скриптовые языки программирования"

## Описание
Веб-сайт, посвященный методам оптимизации скорости загрузки веб-страниц.

## Технологии
- React 19
- Redux Toolkit
- React Router
- Vite
- CSS Modules

## Установка
```bash
npm install
