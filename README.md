# Оптимизация веб-ресурсов

Курсовой проект по дисциплине "Скриптовые языки программирования"

## Описание
Интерактивный учебный веб-сайт, систематизирующий современные методы оптимизации скорости загрузки веб-страниц. Проект демонстрирует практическое применение техник оптимизации в собственной реализации на стеке React + Redux.

## Технологии
- React 19
- Redux Toolkit
- React Router
- Vite
- CSS Modules

## Установка и запуск
```bash
npm install
npm run dev
```

## Структура проекта
```bash
├── src/
│   ├── components/    # Переиспользуемые UI-компоненты
│   ├── pages/         # 6 тематических страниц
│   ├── store/         # Redux конфигурация и слайсы
│   ├── hooks/         # Пользовательские React хуки
│   └── styles/        # Глобальные стили
├── public/
│   ├── data/          # XML конфигурация меню
│   └── icons/         # SVG иконки навигации
└── package.json       # Зависимости проекта
```

##Функциональность
- Динамическое меню с загрузкой из XML
- Адаптив с поддержкой мобильных устройств
- Навигационные карточки для перехода между темами
- Управление состоянием через Redux с асинхронными экшенами

##Тематические разделы
1. Главная - обзор методов оптимизации
2. Загрузка - уменьшение HTTP-запросов, сжатие кода
3. Кэширование - заголовки кэша, серверная настройка
4. Сеть и сервер - CND, Gzip-сжатие, оптимизация сервера
5. Рендеринг - оптимизация загрузки CSS/JavaScript
6. Задержка - минимизация редиректов, DNS оптимизация
